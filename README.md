# Coding Agent Plugin

Multi-agent coding assistant with orchestration capabilities.

## Features
- Orchestration-based agent routing
- Multiple modes: Task, Planning, Quick
- MinIO or local storage support
- Contextual session management

## Installation
```bash
pip install coding-agent-plugin
```


## Quick Start
coding-agent --openapi-url https://api.openai.com/v1
--model gpt-4
--api-key your-key
--mode task
--project-id my-project
